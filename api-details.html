API Key Setup Guide

What is an API key?

In Pro mode, AI Hive sends your prompts directly to each AI and fills in the responses automatically — no copy/paste, no tab switching. An API key is a private password that lets AI Hive talk to each AI on your behalf.

Without a key, AI Hive works in Free mode — you copy/paste prompts manually. API keys are only needed for the fully automated Pro experience.

Your keys never leave your device

Keys are stored in your browser only. They go directly from your browser to each AI provider. AI Hive has no server — your keys are never shared with any third party.

Free or low-cost AIs
Gemini (Google)
Free tier available
Model:gemini-2.0-flashGoogle's fast multimodal model
  1. Go to aistudio.google.com/apikey and sign in with your Google account.
  2. Click Create API Key and copy the key shown.
  3. Paste it into the Gemini field in AI Hive and press Enter.
Gemini is completely free — no credit card required. A generous number of requests per day at no cost. Most AI Hive sessions stay within the free limit easily.
DeepSeek
Very affordable
Model:deepseek-chatDeepSeek's flagship model
  1. Go to platform.deepseek.com/api_keys and sign up.
  2. Click Create new API key and copy it.
  3. Paste it into the DeepSeek field in AI Hive and press Enter.
DeepSeek is 10-20x cheaper per token than OpenAI or Anthropic. Add credit at platform.deepseek.com → Top Up — $5 gets you a lot of usage. Excellent Builder choice on a budget.
Copilot (Microsoft)
Not yet available for individuals
Status:API in Private Preview — enterprise only as of early 2026
Copilot API is not currently available for personal Microsoft 365 subscribers. It requires an enterprise license and is still in Private Preview. This is a Microsoft restriction — not an AI Hive limitation.
Copilot works great in free/manual mode. Open copilot.microsoft.com in a tab, paste your prompts, and copy responses back into AI Hive.
Pay-as-you-go AIs
ChatGPT (OpenAI)
Pay-as-you-go
Model:gpt-4oOpenAI's flagship model
  1. Go to platform.openai.com/api-keys and sign in.
  2. Click Create new secret key, give it a name, click Create.
  3. Copy the key immediately — it is shown only once.
  4. Paste it into the ChatGPT field in AI Hive and press Enter.
Your ChatGPT subscription does NOT include API access — the API is completely separate and pay-as-you-go. Go to platform.openai.com → Billing and add credit ($5–$10 gets you plenty of testing). Without this, all API calls will fail immediately.
Claude (Anthropic)
Pay-as-you-go
Model:claude-opus-4-5Anthropic's capable everyday model
  1. Go to console.anthropic.com/settings/keys and sign in.
  2. Click Create Key, give it a name (e.g. "AI Hive"), click Create.
  3. Copy the key immediately — shown only once.
  4. Paste it into the Claude field in AI Hive and press Enter.
Requires credit balance. Go to console.anthropic.com → Billing and add credit (~$5 minimum). You are only charged for what you use.
Grok (xAI)
Limited availability
Model:grok-3
  1. Go to console.x.ai and sign in with your X account.
  2. Navigate to API Keys and generate a new key.
  3. Paste it into the Grok field in AI Hive and press Enter.
Requires credit balance — add at console.x.ai → Billing. API availability may vary — check the console for current status.
Perplexity
Pay-as-you-go
Model:sonar-pro
  1. Go to perplexity.ai/settings/api and sign in.
  2. Click Generate under API Keys and copy the key.
  3. Paste it into the Perplexity field in AI Hive and press Enter.
Requires credit balance — add at perplexity.ai → Settings → API → Add Credits.
